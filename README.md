Установка зависимостей
pip3 install fastapi uvicorn pytest coverage locust

Запуск сервера
python3 -m uvicorn src.main:app --reload

Запуск тестов
python3 -m pytest

Покрытие тестами
python3 -m coverage run --source=src -m pytest
python3 -m coverage html

Отчёт доступен в папке htmlcov/index.html

Нагрузочное тестирование
python3 -m locust

Открыть в браузере:
http://localhost:8089

В ходе нагрузочного тестирования наблюдаются ошибки для операций PUT и DELETE что связано с использованием фиксированного id (1), который может отсутствовать в базе данных в момент запроса
Операции создания и получения задач работают стабильно без критических ошибок

Тесты используют SQLite базу данных. Данные не изолируются между тестами, однако корректная работа обеспечивается за счет динамического получения id

Кэширование в проекте не реализовано, поэтому влияние на производительность не оценивалось
Нагрузочное тестирование проводилось для оценки базовой производительности API
